# E-Commerce-Analytics-sql

# Project Overview
A SQL-based E-Commerce Analytics project designed to analyze real-world business data and generate meaningful insights such as sales performance, customer behavior, and revenue trends.  This project uses relational database design with multiple interconnected tables including Customers, Orders, Products, Order Items, and Payments .
